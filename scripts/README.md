# Ferocia code exercise

## Installation

Note: All commands are run from the root of the repo.

This repo runs using [pnpm](https://pnpm.io/).

Install packages (prerequisite):

```
pnpm i
```

To run the Term Deposit script:

```
pnpm run:term-deposit
```

To run tests:

```
pnpm test
```

To run linting:

```
pnpm lint
```
